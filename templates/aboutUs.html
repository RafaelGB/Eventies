    Sobre nosotros

    Eventies es un proyecto universitario que trata de facilitar la gestión
    de tu tiempo libre aprendiendo de tus gustos

    Una ventana a nuevas experiencias

    descubre nuestro sistema de recomendaciones

    Eventies te premite crear tus propios planes además de una potente herramienta de búsqueda para
    encontrar los que más te puedan interesar.

    • Filtros como distancia o presupuesto.
    • Mapa con guía de como llegar.
    • Control de asistencia e interesados.

    A medida que te diviertas y vayas asistiendo a dihos eventos nosotros aprenderemos y te recomendaremos los planes más afines a ti.

    Tecnologías usadas

    Bootstrap

    librería de diseño que permite compatibilizar visión web y móvil.

    Django

    Framework web basado en python para front y back.

    posgreSQL

    Base de datos SQL con soporte para datos geométricos.

    Sklearn

    Librería machine learning basada en python.
